# Шахматы

Видео - https://drive.google.com/file/d/1rrt_Z1jA2fZ8n3KG0QEfg0SPVQAQLXet/view?usp=sharing

# Реализовано:
- Игра в шахматы против бота.
- Таймер.
- Просмотр игры бота против бота.
- Режимы игры.

# Запуск:
В терминале выбирается режим игры: 
1) "Хочу играть! [аргумент]" Интерактивная игра против бота. Аргументы: white|black|белый|черный
2) "Я наблюдатель" Просмотр игры бота против бота.

# Правила:
1) Игрок должен объявить вражескому королю шах и мат за 5 минут.
2) По истечении таймера объявляется ничья с возможностью начать заново или закончить игру.
